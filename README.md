# My Journey — Cinematic Portfolio Website

A storytelling-driven cinematic portfolio built using React, Tailwind CSS, and Framer Motion.

This project is not a traditional portfolio focused only on projects and skills. Instead, it is designed as an immersive web experience that visually represents my journey, struggles, failures, discipline, and growth through fullscreen storytelling and motion-driven UI.

## Live Demo

https://journey-page-five.vercel.app/

---

# Features

- Cinematic fullscreen storytelling
- Smooth scroll transitions
- Motion animations using Framer Motion
- Responsive design for desktop and mobile
- Dark immersive UI
- Typography-focused layouts
- Personal visual assets integrated into the experience
- Modern React component architecture

---

# Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- JavaScript

---

# Design Goals

The goal of this project was to explore:
- storytelling through frontend development,
- emotional user experiences,
- cinematic web interactions,
- and modern UI/UX experimentation.

Instead of building a conventional developer portfolio, this project focuses on creating an emotional and memorable browsing experience.

---

# Inspiration

The design inspiration came from:
- cinematic storytelling websites,
- interactive portfolios,
- Apple-style immersive product pages,
- documentary-inspired visual pacing,
- and motion-driven user experiences.

---

# Challenges Faced

Some major challenges while building this project:
- balancing storytelling with readability,
- creating smooth scroll experiences,
- maintaining responsiveness across devices,
- optimizing animations,
- and designing immersive sections without overwhelming the user.

- Responsive layouts

- Visual hierarchy
- User engagement through interaction design
